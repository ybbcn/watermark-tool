### 3.2 并发测试

使用 Locust 进行压力测试：

```python
# locustfile.py
from locust import HttpUser, task, between
import io
from PIL import Image

def generate_test_image():
    img = Image.new('RGB', (800, 600), color='red')
    buffer = io.BytesIO()
    img.save(buffer, format='JPEG')
    buffer.seek(0)
    return buffer.read()

class WatermarkUser(HttpUser):
    wait_time = between(1, 3)
    
    @task(3)
    def add_watermark(self):
        files = {'file': ('test.jpg', generate_test_image())}
        data = {'text': 'Test', 'position': 'bottom-right', 'opacity': '0.5'}
        self.client.post('/api/add-watermark', files=files, data=data)
    
    @task(1)
    def remove_watermark(self):
        files = {'file': ('test.jpg', generate_test_image())}
        data = {'mask_type': 'auto'}
        self.client.post('/api/remove-watermark', files=files, data=data)
```

**测试场景：**

| 并发用户 | 持续时间 | 目标成功率 | 目标 P95 |
|---------|---------|-----------|---------|
| 10 | 5 分钟 | >99% | <3 秒 |
| 50 | 10 分钟 | >95% | <5 秒 |
| 100 | 10 分钟 | >90% | <8 秒 |
