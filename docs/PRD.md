## 1. 产品概述

### 1.1 产品定位

在线水印处理工具，提供**添加水印**和**移除水印**两大核心功能，采用内存处理模式，图片不存储到服务器，保护用户隐私。

### 1.2 产品愿景

让每个人都能简单、快速、安全地处理图片水印。

### 1.3 MVP 目标

用最小成本验证核心功能的市场需求，2 周内上线可用版本。

### 1.4 成功指标

| 指标 | 目标值 | 测量方式 |
|-----|-------|---------|
| 日活跃用户 | 100+ | 访问日志 |
| 图片处理量 | 500+/天 | API 统计 |
| 平均处理时间 | <3 秒 | 后端日志 |
| 用户留存率 | 30%+ | 7 日回访 |

## 2. 目标用户

### 2.1 用户画像

| 类型 | 描述 | 使用场景 |
|-----|------|---------|
| 📸 摄影爱好者 | 个人摄影师、爱好者 | 给作品添加签名水印 |
| 🛒 电商卖家 | 淘宝/拼多多店主 | 批量给商品图加品牌水印 |

### 2.2 用户需求

**添加水印：**
- 快速批量添加
- 自定义位置和透明度
- 支持文字和 Logo

**移除水印：**
- 一键智能移除
- 处理效果自然
- 支持复杂背景

## 3. 核心功能

### 3.1 功能列表

| 优先级 | 功能 | 描述 | 工期 |
|-------|------|------|-----|
| P0 | 添加文字水印 | 支持位置、透明度、字体大小 | 2 天 |
| P0 | 添加 Logo 水印 | 上传图片 Logo，支持缩放 | 1 天 |
| P0 | 移除水印（简化版） | 自动检测高亮区域移除 | 2 天 |
| P0 | 图片预览 | 上传后实时预览 | 1 天 |
| P0 | 结果下载 | 处理完成后下载 | 0.5 天 |
| P1 | 批量处理 | 一次上传多张图片 | 2 天 |
| P1 | 自定义蒙版 | 手动标注移除区域 | 2 天 |
| P2 | 用户系统 | 登录、历史记录 | 3 天 |
| P2 | 付费功能 | 高级去水印、API 调用 | 3 天 |

### 3.2 功能详细说明

#### 3.2.1 添加文字水印 (P0)

**用户流程：**
```
上传图片 → 输入文字 → 选择位置 → 调整透明度 → 预览 → 下载
```

**功能要求：**
- 支持常见图片格式：JPG, PNG, WebP
- 最大支持 10MB 图片
- 位置选项：左上、右上、左下、右下、居中
- 透明度：10% - 100% 可调
- 字体大小：自适应图片尺寸
- 处理时间：<2 秒

**技术实现：**
- Python Pillow 库
- 内存处理，不落地存储

---

#### 3.2.2 添加 Logo 水印 (P0)

**用户流程：**
```
上传原图 → 上传 Logo → 选择位置 → 调整大小 → 预览 → 下载
```

**功能要求：**
- Logo 支持 PNG 透明背景
- Logo 大小：原图的 5% - 50% 可调
- 位置选项：同上
- 处理时间：<2 秒

---

#### 3.2.3 移除水印 (P0)

**用户流程：**
```
上传图片 → 自动检测 → AI 处理 → 预览 → 下载
```

**功能要求：**
- MVP 版本：自动检测高亮区域（白色水印）
- 进阶版本：用户手动标注区域
- 处理时间：<5 秒

**技术实现：**
- MVP: OpenCV inpaint
- 进阶：LaMa 大模型

---

#### 3.2.4 图片预览 (P0)

**功能要求：**
- 上传后立即显示缩略图
- 支持放大查看细节
- 处理前后对比（滑动对比）

---

#### 3.2.5 结果下载 (P0)

**功能要求：**
- 处理完成后显示下载按钮
- 自动命名：`watermarked_原文件名.jpg`
- 支持直接下载或复制到剪贴板

## 4. 技术方案

### 4.1 技术架构

```
┌─────────────────────────────────────────┐
│              用户浏览器                  │
│           (HTML + CSS + JS)             │
└─────────────────┬───────────────────────┘
                  │ HTTP/HTTPS
┌─────────────────▼───────────────────────┐
│              前端服务器                  │
│         (Nginx / Cloudflare)            │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│              后端 API                     │
│         (FastAPI + Uvicorn)             │
│  ┌─────────────────────────────────┐   │
│  │  添加水印  │  移除水印  │ 其他   │   │
│  └─────────────────────────────────┘   │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│           图片处理引擎                   │
│      (Pillow + OpenCV + NumPy)          │
│         内存处理，不存储文件             │
└─────────────────────────────────────────┘
```

### 4.2 技术栈

| 层级 | 技术 | 选型理由 |
|-----|------|---------|
| 前端 | HTML + TailwindCSS | 快速开发，无需构建 |
| 后端 | Python 3.11 + FastAPI | 开发效率高，生态丰富 |
| 图片处理 | Pillow + OpenCV | 成熟稳定，功能全面 |
| 容器化 | Docker + Compose | 一键部署，环境隔离 |
| 国内部署 | 腾讯云服务器 | 访问速度快 |
| 国外部署 | Cloudflare + Railway | 免费额度够用 |

### 4.3 API 设计

#### 添加文字水印

```http
POST /api/add-watermark
Content-Type: multipart/form-data

参数:
- file: 图片文件 (必填)
- text: 水印文字 (必填)
- position: 位置 (可选，默认 bottom-right)
- opacity: 透明度 0-1 (可选，默认 0.5)

响应:
- 成功：图片文件流 (image/jpeg)
- 失败：{"detail": "错误信息"}
```

#### 添加 Logo 水印

```http
POST /api/add-logo-watermark
Content-Type: multipart/form-data

参数:
- file: 原图 (必填)
- logo: Logo 图片 (必填)
- position: 位置 (可选)
- scale: 大小比例 0-1 (可选，默认 0.2)

响应：同上
```

#### 移除水印

```http
POST /api/remove-watermark
Content-Type: multipart/form-data

参数:
- file: 图片文件 (必填)
- mask_type: 处理模式 (可选，auto/manual)

响应：同上
```

## 5. 部署方案

### 5.1 国内版（腾讯云）

**服务器配置：**
| 配置 | 规格 | 费用 |
|-----|------|------|
| CPU | 2 核 | ¥70/月 |
| 内存 | 4GB | 包含 |
| 带宽 | 3Mbps | 包含 |
| 系统 | Ubuntu 22.04 | 免费 |

**部署步骤：**

```bash
# 1. 购买服务器后，SSH 登录
ssh root@your-server-ip

# 2. 安装 Docker
curl -fsSL https://get.docker.com | sh

# 3. 上传项目
git clone https://github.com/your-repo/watermark-tool.git
cd watermark-tool

# 4. 一键部署
./deploy/deploy.sh

# 5. 配置域名（可选）
# 在腾讯云 DNS 解析域名到服务器 IP
```

**Nginx 配置：**
- 反向代理到 3000（前端）和 8000（API）
- 配置 HTTPS（Let's Encrypt）

## 7. 风险与合规

### 7.1 法律风险

**风险描述：**
用户可能使用工具移除他人版权水印，导致侵权纠纷。

**缓解措施：**
1. 在显著位置添加免责声明
2. 用户协议中明确禁止侵权行为
3. 设置举报机制
4. 保留违规用户封禁权利

**免责声明文案：**

```
⚠️ 法律声明

本工具仅供学习和个人使用。请勿用于移除他人拥有版权的水印，
由此产生的法律后果由使用者自行承担。

使用本工具即表示您同意：
1. 仅对拥有版权或授权的图片进行处理
2. 不将处理结果用于商业侵权
3. 遵守当地法律法规
```

---

### 7.2 技术风险

| 风险 | 概率 | 影响 | 缓解措施 |
|-----|------|------|---------|
| 大图片内存溢出 | 中 | 高 | 限制最大 10MB |
| 并发过高服务崩溃 | 中 | 高 | 限流 + 队列 |
| 去水印效果不佳 | 高 | 中 | 明确说明简化版 |
| 服务器被攻击 | 低 | 高 | 防火墙 + 限流 |

---

### 7.3 合规要求

**中国大陆：**
- ICP 备案（如需域名访问）
- 公网安备（部分地区要求）
- 用户协议 + 隐私政策

**海外：**

- DMCA 响应机制
